CSS in Hindi | CSS क्या है हिन्दी मे |

CSS वह भाषा है जिसका उपयोग web page पर HTML सामग्री को style करने अर्थात उसे सुन्दर बनाने के लिए किया जाता है । CSS बताता है कि HTML element को web page पे कैसे प्रदर्शित किया जाना चाहिए । एक बात ध्यान दे की CSS सिखने के लिए आपको HTML आना आवश्यक है |

CSS की शुरुआत कब हुई ?

CSS की शुरुआत सबसे पहले 10 अक्टूबर 1994 को Håkon Wium Lie ने किया था | इसके पहले website सिर्फ HTML के बने होते थे जिसमे text सिर्फ काले रंग के होते थे जो notepad मे लिखे हुए text जैसे दीखते थे जिसके कारण website देखने मे बहुत ही खाराब होते थे |

CSS का उपयोग क्यों किया जाता है ?

CSS का उपयोग Web Designing के लिए किया जाता है यानि इसका उपयोग website को दिखने मे सुन्दर बनाने के लिए किया जाता है जिससे website आकर्षक लगे |

CSS के लाभ क्या है ?

आप CSS सीख कर एक Professional Web Designer बन सकते है यानि आप किसी के भी website को design कर सकते है | CSS के उपयोग द्वारा हम जिस तरह से website का रूप रखना चाहे रख सकते है यानि उसके text के आकर को बढ़ा सकते है , उसमे रंग दे सकते है , उसमे animation कर सकते है | आप जितना अच्छा चाहे अपने अनुसार website को design कर सकते है | सबसे निचे दो website का image दिया गया है जिसमे पहला image सिर्फ HTML द्वारा बनाया गया website का है जो देखने मे अच्छा नहीं है फिर दूसरा image मे जो website है उसमे पहले वाले HTML द्वारा बनाय गए website को CSS से style करके बहुत ही सुन्दर और आकर्षक बनाया गया है | जिसे देखने के लिए निचे दिए हुए button पे click करे |
