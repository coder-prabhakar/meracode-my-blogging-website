CSS Selector in Hindi - पूरी जानकारी |

CSS selector का उपयोग उस HTML element को चुनने के लिए करते है जिसे आप CSS मे style करना चाहते हैं । CSS selector मे HTML element का चयन उसकी id, class, type, attribute आदि के आधार पर किया जाता हैं ।


CSS Selector के प्रकार |

CSS में कई तरह के selector होते हैं । कुछ महत्वपूर्ण तथा ज्यादा उपयोग होने वाले selector के नाम निचे दिए गए है |

  • Element Selector
  • Id Selector
  • Class Selector
  • Universal Selector
  • Group Selector

Element Selector in CSS in Hindi |

इसका मतलब होता है की CSS selector मे element को चुनने के लिए element के नाम का प्रयोग किया जाए जैसे - h1, p, div, span आदि |

Element Selector Example :-

निचे एक उदाहरण दिया गया है जिसमे एक <h1> तथा दो <p> element को उनके element के नाम ( h1 , p ) द्वारा style किया गया है |
जिसमे <h1> element मे लिखे गए text के color को red त्तथा उसे center मे किया गया है और <p> element मे लिखे गए text के color को blue त्तथा उसे justify किया गया है |

Id Selector in CSS in Hindi |

Id selector एक विशेष element को चुनने के लिए element की id attribute का उपयोग करता है ।

एक id हमेशा एक page के अंदर अद्वितीय होती है यानि एक id एक page के अंदर एक ही element की हो सकती है इसलिए id selector का उपयोग अद्वितीय element को चुनने के लिए किया जाता है |

Id selector मे element को चुनने के लिए पहले # लिखे फिर उस element की id को लिखे जो आप style करना चाहते है |

Id Selector Example :-

निचे एक उदाहरण दिया गया है जिसमे एक <h1> तथा दो <p> element को उनके id name ( heading , para1 , para2 ) के उपयोग द्वारा style किया गया है |
जिसमे id नाम heading वाले element मे लिखे गए text के color को red त्तथा उसे center मे किया गया है और id नाम para1 तथा para2 वाले element मे लिखे गए text के color को blue त्तथा उसे justify किया गया है |

Class Selector in CSS in Hindi |

Class selector एक element को चुनने के लिए element के class attribute का उपयोग करता है ।

Class selector मे element को चुनने के लिए पहले (.) लिखे फिर उस element के class का नाम लिखे जो आप style करना चाहते है |

Class Selector Example :-

निचे एक उदाहरण दिया गया है जिसमे एक <h1> तथा दो <p> element को उनके class name ( heading , paragraph ) के उपयोग द्वारा style किया गया है |
जिसमे class नाम heading वाले element मे लिखे गए text के color को red त्तथा उसे center मे किया गया है और class नाम paragraph वाले element मे लिखे गए text के color को blue त्तथा उसे justify किया गया है |

Universal Selector in CSS in Hindi |

Universal selector का उपयोग एक ही बार मे page पर के सभी HTML element को चुनने के लिए किया जाता है |

इसका उपयोग करने के लिए selector की जगह * लिखा जाता है जिससे page के सभी element का चयन हो जाता है |

Universal Selector Example :-

निचे एक उदाहरण दिया गया है जिसमे एक <h1> तथा दो <p> element को एक ही बार मे style करने के लिए universal selector * का उपयोग किया गया है |
जिसमे सभी element के text के color को orangered त्तथा उसे center मे किया गया है |

Group Selector in CSS in Hindi |

Group selector का मतलब होता है की बहुत सारे element को उनके अलग - अलग element name , class name या id name को एक ही बार मे चयन करके style करना |

Group selector मे बहुत सारे element को style करने के लिए selector मे अलग - अलग element, class, id के बिच मे (,) देना होता है |

Group Selector Example :-

निचे एक उदाहरण दिया गया है जिसमे एक <h1> तथा दो <p> element को group selector के उपयोग द्वारा एक ही बार मे तीनो element को उनके element के नाम h1 , p को चयन करके style कर दिया गया है |
जिसमे सभी element के text के color को orangered त्तथा उसे center मे किया गया है |
