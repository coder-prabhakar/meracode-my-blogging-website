CSS content Property in Hindi |

CSS मे content property का उपयोग web page पे सामग्री प्रदर्शीत होते समय कोई element के पहले या बाद मे कोई भी चीज (text, images, symbols, emojis etc.) प्रदर्शीत करवाने के लिए किया जाता है |
उदाहरण के लिए - मान लीजिये HTML के किसी element मे आपने कोई text लिखा है और आप चाहते है की जब वह text दिखे web page पे उस समय उस text के पहले या बाद मे कुछ text, images, symbols, emojis आदि भी दिखे तो आप CSS के अंदर content property का उपयोग करके यह कर सकते है |

Note:- कोई element के पहले या बाद मे कुछ भी प्रदर्शीत(display) करवाने के लिए आपको CSS मे उस element के जगह pseudo-element(::before & ::after) का उपयोग करना होगा |
जैसे -   p::after { ........ }   या   p::before { ........ }

Content Property Values in Hindi |

निचे कुछ content property मे उपयोग होने वाले अलग - अलग प्रकार के value के बारे मे जानकारी दि गयी है |

Value Value के बारे मे जानकारी

Textआप content property के सहायता से web page मे text भी प्रदर्शीत करवा सकते है |
Syntax -  p::before { content: "Student Name -"; }

Numberआप content property के सहायता से web page मे Number भी प्रदर्शीत करवा सकते है |
Syntax -  p::after { content: "100"; }

ImageImage प्रदर्शीत करवाने के लिए आपको content property के value मे url() का उपयोग करके इसके अंदर image का url देना होगा |
Syntax -  p::after { content: url(Image/show-small-image.jpg); }

symbolआप इसका उपयोग करके symbol या emoji भी प्रदर्शीत करवा सकते है |
Syntax -  p::before { content: "⛅"; }


Content Property Example in Hindi |

ऊपर आपने जितने भी content property के value के बारे मे पढ़ा उन सबको अब हम example(उदाहरण) के माध्यम से समझेंगे |

निचे एक web page का code दिया गया है जिसमे ऊपर बताये गए content property के सारे value का इस्तेमाल किया गया |
आप चाहे तो इस code के निचे एक copy का button दिया गया है जिसपे आप click करके सारे code को copy कर सकते है और उसे अपने browser मे खोलकर देख सकते है |

ऊपर जो code दिया गया है उससे बना web page कैसा होगा उसका image निचे दिया गया है जिसे देख कर आप content property के बारे मे अच्छे से समझ पाएंगे की ये कैसे काम करता है |
