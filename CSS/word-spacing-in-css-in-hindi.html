Word Spacing in CSS in Hindi |

word-spacing property का उपयोग कोई वाक्य के प्रत्येक शब्द के बिच मे खाली स्थान बढ़ाने या घटाने के लिए यानि कोई sentence या paragraph के प्रत्येक word के बीचे मे कितना space रहेगा उसे बताने के लिए किया जाता है |

Example के लिए मान लिजिये की आपने कोई element मे कुछ word का paragraph लिखा है आप चाहते है की जब वह paragraph browser मे दिखे यानि website मे दिखे तो उस paragraph के प्रत्येक word के बिच मे 60px का space हो तो आप css मे word-spacing property की सहायता से यह कर सकते है |

CSS word-spacing के उदाहरण |

सबसे ऊपर मे एक web page (website) का image दिया हुआ है जिसमे आप देख सकते है की एक ही paragraph को दो बार लिखा गया है जिसमे से पहले वाले paragraph मे कोई ज्यादा(extra) space नहीं दिया गया है लेकिन दूसरा वाले paragraph मे प्रत्येक word के बिच 60px का space दिया गया है जिससे उसमे के सारे word एक-दुसरे से काफी दुरी पर दिख रहे है |

ऊपर जो web page का image दिया गया है उसका code निचे दिया हुआ है जिसे देख कर आप word-spacing property के बारे मे अच्छे से समझ सकते है तथा इस code के निचे एक copy का button दिया गया है जिसपे आप click करके इस code को copy कर सकते है |

निचे दिए हुए code मे आप देख पाएंगे की दो <h1> element है और उन दोनों मे एक ही text लिखा हुआ है तथा दुसरे वाले <h1> element का एक class name दिया गया है जो space है | <style> tag के अंदर दुसरे वाले <h1> element को उसके class name से select करके उसमे word-spacing: 60px; किया गया है जिससे उस <h1> element के प्रत्येक word के बिच मे 60px का space उत्पन्न हो जाएगा |
