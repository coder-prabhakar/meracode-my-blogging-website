CSS Syntax in Hindi |

अगर आपको web page के HTML document को style करना है यानि website को css से style करना है तो आपको CSS syntax के बारे मे जानकारी होना आवश्यक है अर्थात HTML को style करने के लिए कुछ भी बदलाव करने के लिए आपको CSS syntax नियम का पालन करना होगा |


CSS लिखने का तरीका हिंदी में |

CSS में style करने के लिए नियम बने होते हैं | एक style नियम तीन भागों से बना होता है -

  • Selector  - selector उस HTML element को कहते है जिसे आप style करना चाहते हैं ।

  • Property  - property का मतलब होता है की आप HTML element को style करने मे जो बदलाव करना चाहते है उस css property का नाम | जैसे- color , font, height, width etc |

  • Value  - इसका मतलब property का value |

अगर h1 element के text के color को green करना है तो css नियम का पालन करते हुए selector की जगह h1 होगा , property की जगह color होगा तथा value की जगह green होगा | निचे आप image मे देख सकते है |

Inline CSS in Hindi

Inline CSS का उपयोग एक अकेला element को style करने के लिए किया जाता है यानि जब आपको सिर्फ एक element को style करना होता है तब इसका उपयोग किया जाता है | Inline CSS style मे हमें जिस element को style करना होता है उस element के अंदर ही style attribute का उपयोग करके उसको style किया है | यह style element मे ही होता है तो इसे style करने के लिए Selector का उपयोग नहीं होता बस Property और Value का होता है |

Internal or Embedded CSS in Hindi

Internal CSS या Embedded CSS एक ही प्रकार के CSS को कहते है बस इनके दो अलग - अलग नाम है Internal या Embedded | Internal CSS का उपयोग एक अकेला web page को style करने के लिए किया जाता है यानि अगर हमें सिर्फ एक web page को style करना होता है तो उसके लिए Internal CSS का उपयोग किया जाता है | इसमें हमें जिस HTML page को style करना होता है उसके <head> section के अंदर <style> tag लगाकर उसमे web page को style करने वाले CSS के code को लिखते है |

Internal CSS Example :-

निचे एक उदाहरण दिया गया है जिसमे Internal or Embedded CSS का प्रयोग किया गया है | दिए गए code मे एक h1 element को <head> section के अंदर <style> tag मे उसको style किया गया है यानि उसके color को red तथा उसके text को center मे किया गया है |


External CSS in Hindi

जब हम HTML page को style करने के लिए एक अलग file मे CSS लिखते है तो उसे External CSS कहते है | External CSS को CSS file मे लिखा जाता है तथा इस file को HTML page से लिंक कर दिया जाता है यानि HTML page से जोड़ दिया जाता है | CSS का उपयोग सबसे ज्यादा External CSS मे होता है तथा इसको CSS उपयोग करने का सबसे अच्छा तरीका माना जाता है | External CSS का उपयोग एक web page या एक से ज्यादा web page को style करने के लिए किया जा सकता है |

CSS File कैसे बनाये ?

CSS code को CSS file के अंदर लिखा जाता है | CSS का code लिखने के लिए आप कोई भी Text editor का उपयोग कर सकते है जैसे- Notepad , TextEdit आदि | आपने जिस file मे CSS का code लिखा है उस file को style.css के नाम से save करे आप चाहे तो अपने file का दूसरा नाम भी रख सकते है लेकिन ध्यान दे की आप जो भी नाम रखे उसके बाद मे .css लिखना आवश्यक है | जैसे - style.css , website.css , design.css आदि |

Note:- कोई भी text editor मे आप जो भी कुछ लिखकर save करेंगे तो वो एक साधारण text file बनेगा उसे CSS file बनाने के आप जो भी file का नाम रख रहे है उसके अंत मे .css लगाकर उस file को save करें |

CSS file को HTML file से कैसे जोड़े ?

CSS file को HTML से जोड़ने के लिए हम link tag का उपयोग करते है | जिसका syntax <link rel="stylesheet" href="style.css"> होता है जिसके href attribute मे हम css file के url को लिखते है | इसे हम HTML के <head> section के अंदर लिखते है |

External CSS Example :-

निचे एक उदाहरण दिया गया है जिसमे HTML code के <head> section के अंदर <link> tag का उपयोग करके External CSS को जोड़ा गया है | दिए गए उदाहरण मे link tag के अंदर href attribute मे css file का url दिया गया है जिसका नाम style.css है |
