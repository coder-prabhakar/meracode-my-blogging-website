CSS !important Property in Hindi |

अगर आपने एक ही element के कोई एक प्रकार के style को एक-से-ज्यादा अलग-अलग जगह style कर दिया है और आप चाहते है की उसमे से कोई एक विशेष style ही उस element पे लागु हो तो आप !important के उपयोग द्वारा यह कर सकते है |

उदाहरण के लिए - अगर आप कोई एक element के text के color को CSS मे कही red, कही green, कही yellow यानि अलग-अलग जगह अलग-अलग रंग दे देते है और आप चाहते है की element के text का color बस green दिखे तो आप !important का उपयोग द्वारा यह कर सकते है बस आपको जहाँ color: green; है वहाँ पे color: green !important; लिखना होगा |
