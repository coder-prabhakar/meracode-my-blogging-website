HTML Editor का क्या उपयोग है ?

HTML code को लिखने मे आसान बनाने के लिए HTML editor का उपयोग किया जाता है क्योकि जब हम text editor मे code लिखते है तो हमें साधरण text की तरह code को लिखना पड़ता है लेकिन जब हम HTML editor मे code को लिखते है तो उसमे हमारे code के गलतियों के बारे मे पाता चलता है, HTML editor मे color theme होते है जिससे code का हर प्रकार का चीझ का अलग - अलग रंग का दिखता है जिससे code पढने मे आसानी होती है और HTML editor मे हमें पूरा code लिखना नहीं पड़ता उसके अंदर बहुत से shortcuts text होते है जिसका उपयोग करके आप code के बरे - बरे भाग को थोरे से text के जरीये लिख सकते है |

HTML Editor क्या है तथा इसके लाभ लिखिए ?

HTML editor एक ऐसा प्रकार का software होता है जिसका उपयोग HTML code को लिखने मे आसान त्तथा सुविधाजनक बनाने के लिए किया जाता है | HTML editor का उपयोग करके आप HTML code को बहुत ही कम समय मे सही - सही लिख सकते है |

HTML Editor के लाभ निम्नलिखित है -

  • इसमें पुरे code को लिखने की आवश्यकता नहीं पड़ती यानि इसमें समय की बहुत बचत होती है |
  • इसमें code लिखना बहुत ही आसान होता है |
  • इसमें हर प्रकार के code का अलग - अलग रंग (color) होता है जिससे code को पढने या समझने मे आसानी होती है |
  • इसमें code लिखते समय अगर आपके code मे कुछ गलती हो जाए तो आपके गलती के बारे मे ये बता देता है |

कोई दो HTML Editors के नाम लिखें ।

Code लिखने के लिए बहुत से software उपलब्ध है जिसमे से ज्यादा तर उपयोग होने वाले त्तथा सबसे अच्छे दो code editor के नाम निम्नलिखित है |

  • Sublime Text.

HTML coding के लिए कौन सा software चाहिए ?

HTML coding करने के लिए सबसे अच्छा software मे से एक Visual Studio Code को माना जाता है | HTML coding के लिए सबसे ज्यादा उपयोग होने वाला software भी Visual Studio Code ही है इसका उपयोग आप HTML के अलावा CSS, JavaScript और भी बहुत से language के coding करने के लिए कर सकते है |

Text editor मे code लिखते समय अगर कुछ गलती होता है तो text editor हमें उसके बारे मे सुचना नहीं देता यानि हमें गलतियों के बारे मे पाता नहीं चलता |

अगर HTML editor मे code लिखते समय हमसे कुछ गलती हो जाए तो उसके बारे मे HTML editor हमें सुचना दे देती है की हमसे गलती हुई है |

Text editor मे हमें जितना code लिखना होता है तो हमें उस पुरे code को लिखना पड़ता है जिसमे बहुत समय लगता है |

लेकिन अगर हमें HTML editor मे code लिखना होता है तो हम shortcuts का उपयोग करके बरे - बरे code को बहुत कम समय मे लिख सकते है |


HTML Editor कैसे काम करता है ?

HTML editor मे आप कोई भी tag या element को जैसे ही लिखना शुरू करते है तो editor उससे मिलता - जुलता tag या element दिखाने लगता है अगर आप जो लिख रहे है editor उसी को दिखा रहा है तो आप जैसे ही enter का button दबाते है तो वह tag या element का पूरा syntax लिखा जाता है यानि आप कोई भी element या tag का सिर्फ नाम लिख कर enter का button दबाते है तो HTML editor मे उस element या tag का पूरा syntax लिखा जायेगा |
जैसे - अगर आप h1 element को लिखना चाहते है तो text editor मे आपको उसका पूरा syntax लिखना होगा जो <h1></h1> होता है लेकिन अगर आप वही h1 element को HTML editor मे लिखते है तो आप सिर्फ h1 लिख कर enter button दबाते है तो <h1></h1> लिखा जायेगा |
