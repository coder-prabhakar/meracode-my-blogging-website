HTML Text tag in Hindi |

HTML मे text को लिखने के लिए बहुत सारे tags होते है जिसमे से मुख्य रूप से ज्यादा तर <h1> तथा <p> का उपयोग होता है और भी Text को अलग - अलग तरीको से लिखने के लिए tag होते है जिन्हें हम Formatting tags कहते है |

Formatting tags in HTML in Hindi |

HTML मे Formatting tags का उपयोग text को web page पर कुछ अलग तरीको से दिखाने के लिए किया जाता है ज्यादा तर हम heading के लिए <h1> tag त्तथा paragraph लिखने के लिए <p> tag का उपयोग करते है लेकिन जब हमें text को bold, small, italic त्तथा text को Underline करना होता है तो उसके लिए हम Formatting tags का उपयोग करते है |

निचे कुछ Formatting tags के उदाहरण दिए गए है -
<b> , <i> , <small> , <u> , <strong> , <mark> , <ins> , <del> , <dfn> , <strong> , <em> , <abbr> , <address> आदि |

HTML Bold Tag in Hindi |

HTML मे text को bold लिखने के लिए हम <b> tag का उपयोग करते है | यह text को bold कर देता है यानि text को मोटा बना देता है | इसका उपयोग किसी मह्त्वपूर्ण बात, नाम, शहर, आदि को प्रदर्शीत करने के लिए होता है |
इसका syntax :- <b>......................</b> होता है |

HTML <b> tag Example :-

निचे एक उदाहरण दिया गया है जिसमे आप देख सकते है की पहले <p> element मे बहुत सारे text लिखे हुए है जिसमे से एक word (शब्द) को <b> tag के अंदर लिखा गया है जिससे वह Output मे bold दिख रहा है |

HTML Small Tag in Hindi |

HTML मे text को small करने के लिए हम <small> tag का उपयोग करते है | यह text को small कर देता है यानि text को छोटा कर देता है |

HTML <small> tag Example :-

निचे एक उदाहरण दिया गया है जिसमे आप देख सकते है की पहले <p> element के अंदर कुछ text लिखा हुआ है फिर उसी text को <small> tag के अंदर भी लिखा गया है | आप HTML code के निचे उसके Output मे देख सकते है की <small> tag मे लिखा गया text <p> element मे लिखे गए text से छोटा दिख रहा है |

HTML Italic Tag in Hindi |

अगर हमें HTML मे text को italic मे लिखना होता है तो उसके लिए हम <i> tag का उपयोग करते है |

HTML <i> tag Example :-

निचे एक उदाहरण दिया गया है जिसमे <p> element के अंदर कुछ text लिखा हुआ है फिर उसी text को <i> tag के अंदर भी लिखा गया है | आप निचे Output मे देख सकते है की <p> element मे लिखे गए text साधारण दिख रहे है लेकिन उसी text को <i> tag मे लिखा गया है तो वे italic दिख रहे है |

Note:- italic text साधारण text से कुछ टेढ़ा दीखते है |

HTML Underline Tag in Hindi |

HTML मे text को underline करने के लिए हम <u> tag का उपयोग करते है यानि <u> tag का उपयोग करने से text के निचे एक सीधी line (रेखा) आ जाती है |

HTML <u> tag Example :-

निचे एक उदाहरण दिया गया है जिसमे <p> element के अंदर कुछ text लिखा हुआ है फिर उसी text को <u> tag मे भी लिखा गया है | आप निचे Output मे देख सकते है की <p> element मे लिखे गए text साधारण दिख रहे है लेकिन <u> tag मे लिखे गय text के निचे एक सीधी line आ गयी है |

HTML Strong Tag in Hindi |

<strong> tag मे लिखा गया text <b> tag मे लिखा गया text जैसा ही bold दिखता है यानि मोटा दिखता है बस फर्क इतना है की इसका उपयोग करने से text तो bold दिखता ही है साथ मे google को <strong> tag ये भी बताता है की यह text महत्वपूर्ण है |

HTML <strong> tag Example :-

निचे एक उदाहरण दिया गया है जिसमे आप देख सकते है की पहले <p> element मे बहुत सारे text लिखे हुए है जिसमे से एक word (शब्द) को <strong> tag के अंदर लिखा गया है जिससे वह Output मे bold दिख रहा है |
