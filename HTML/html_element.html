HTML के Element क्या है समझाइये ?

हमें web page पे अगर कुछ प्रदर्शीत करवाना होता है तो उसके लिए हम HTML code मे element का उपयोग करते है | एक web page बहुत सारे element से मिलकर बना होता है जिसमे अलग अलग element का काम अलग - अलग है | एक element तीन चिझो से मिलकर बना होता है जो Opening tag उसका content तथा Closing tag होता है |

जैसे - <tagname>Content यहाँ लिखा जाता है....</tagname>

HTML में Element का उदाहरण बताइए ?

निचे कुछ HTML मे ज्यादा तर प्रयोग होने वाले element का उदाहरण दिया गया है ( जैसे- h1, p, ul ) | आप तो जानते ही होंगे की एक element तीन चिझो से मिलकर बना होता है जो Opening tag उसका content तथा Closing tag होता है निचे दिए हुए उदाहरण को देखकर आप इसके बारे मे अच्छे से समझ पाएँगे |

  • <h1>It is used for heading</h1>
  • <p>It is used for paragraph</p>
  • <ul>
  • <li>First Product</li>
  • <li>Second Product</li>

अभी तक जितना HTML विकशित हुआ है उसमे 110 element है जिसमे tags तथा empty element भी शामिल है | इस 110 element मे 32 ऐसे element है जो HTML का नया version जिसे HTML5 कहते है उसके आने के बाद जोड़े गए है |

HTML के Basic Element को समझाइए |

निचे HTML के प्रमुख तथा basic element के नाम और उनका कार्य के बारे मे बताया गया है |

<html> :- यह HTML का मुख्य element है | HTML code मे पहला element यही आता है बाकि सारे HTML element इसके अंदर ही होते है |

<head> :- <html> element के अंदर दो ही element होते है जिसमे से पहला <head> element है इसमें web page की मुख्य चिझे शामिल होती है | जैसे- Page का title , Page का icon आदि |

<body> :- <html> element के अंदर दूसरा element <body> होता है | <body> element के अंदर वो सारे element होते है जिनका काम content को web page पे प्रदर्शीत करवाना है |

<h1> :- <h1> element का ज्यादा तर उपयोग कोई article या paragraph का Heading यानि शीर्षक लिखने के लिए होता है |

<p> :- <p> element का मुख्य उपयोग paragraph यानि कोई text लिखने के लिए होता है |

<li> :- इसका उपयोग किसी बात को list मे दिखाने के लिए किया जाता है |

<div> :- इसका उपयोग बहुत सारे element को एक समूह मे करने के लिए किया जाता है |

HTML Empty Element in Hindi |

कुछ ऐसे element होते है जिसमे closing tag नहीं होता जिसे empty element बोलते है | इस प्रकार के element मे content नहीं लिखा जाता इनका उपयोग कुछ मुख्य कार्यो के लिए किया जाता है |
जैसे- <br>, <hr>
