HTML Syntax in Hindi |

HTML का code लिखने के लिए हमें HTML syntax नियम का पालन करना होगा यानि HTML लिखने के तरीके को समझना होगा | HTML मे बहुत सारे element और tag होते है जिनका अलग - अलग काम है तथा उनका उपयोग भी अलग - अलग जगह पे किया जाता है | निचे उसी के बारे मे पूरा जानकारी दिया गया है जिसे पढने के बाद आप HTML लिखने के basic तरीके को समझ जायेंगे |


HTML Syntax Rules |

HTML code मे सबसे पहले जो tag लिखा जाता है वो <html> tag होता है इसके अंदर दो tag आते है जो <head> और <body> होते है | Web page पे हमें जो कुछ भी दिखता है वो सारे <body> tag के अंदर लिखे जाते है | <body> tag के अंदर बहुत सारे element होते है जिनका काम web page पे अलग - अलग सामग्री को अलग - अलग तरीके से प्रदर्शीत करना है |


HTML Basic tags in Hindi |

HTML के कुछ basic tag निचे दि हुई है |

  • <html>
  • <head>
  • <body>
  • <title>
  • <h1>
  • <p>

Head tag in HTML in Hindi |

<head> tag के अंदर भी बहुत सारे tag आते है जिसमे मुख्य रूप से <title>, <link>, <meta> आदि होते है, एक बात ध्यान दे की <head> tag मे जो भी कुछ लिखा जाता है वो website पे नहीं दिखता यानि web page पे प्रदर्शीत नहीं होता | इसमें जो tag होते है उनका काम <body> tag के element से बहुत अलग होता है |

Head content in Hindi |

<head> tag मे क्या - क्या होता है यानि उसमे किस प्रकार का code लिखा जाता है उसके बारे मे कुछ निम्नलिखित बात निचे दि गई है |

  • इसके अंदर एक <title> tag होता है जिसका उपयोग web page का title लिखने के लिए किया जाता है |
  • इसके अंदर एक <link> tag होता है जिसका उपयोग External CSS को HTML से जोड़ने के लिए किया जाता है |
  • इसके अंदर बहुत सारे <meta> tag होते है जिसका अलग - अलग काम होता है जिसमे से एक <meta> tag का काम web page मे जो content है उसका language यानि भाषा search engine को बताने के लिए किया जाता है |
  • <head> tag के अंदर और भी बहुत प्रकार के tag होते है |

Body tag in HTML in Hindi |

Web page मे हमें जो कुछ भी content , image , video , audio आदि दिखता है उन सारे का code <body> tag के अंदर element मे लिखे होते है | <body> tag के अंदर बहुत से element होते है जिनका काम अलग - अलग प्रकार के सामग्री को web page पे प्रदर्शीत करना होता है, जैसे - text लिखने के लिए , कोई image प्रदर्शीत करने के लिए , कोइ video या audio दिखाने के लिए , किसी content को list मे दिखाने के लिए , कोई भी प्रकार का form बनाने के लिए और भी बहुत कुछ |

Body content in Hindi |

<body> tag के अंदर उपयोग होने वाले कुछ महत्वपूर्ण element के नाम और उनका कार्य निम्नलिखित है |

  • <h1> element का उपयोग कोई article का heading यानि उसका शीर्षक लिखने के लिए होता है |
  • <p> element का उपयोग कोई article या कोई text लिखने के लिए किया जाता है |
  • <li> element का उपयोग कोई चीझ को list मे दिखाने के लिए किया जाता है |


Element के बारे मे पूरी जानकारी लेने के लिए निचे दिए हुए button पे click करे |
