HTML Id

HTML id attribute का उपयोग एक element का अद्वितीय id देने के लिए किया जाता है | HTML मे एक ही id वाले एक से अधिक element नहीं हो सकते है | Element को id एक विशेष पहचान देता है जिसका उपयोग CSS और JavaScript द्वारा कुछ कार्यों को करने के लिए किया जाता है ।

Using id attribute

JavaScript मे id द्वारा किसी element तक पहुचने और उसमे हेरफेर करने के लिए getElementById() method का उपयोग किया जाता है |

CSS मे id द्वारा किसी element तक पहुचने और उसमे हेरफेर करने के लिए CSS के अंदर पहले # लिखे, उसके बाद id name लिखे फिर curly braces {} के अंदर CSS properties को यानि element मे जो बदलाव करना है वो लिखे |
