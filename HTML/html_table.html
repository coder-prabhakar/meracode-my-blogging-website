HTML Table Tags


Tag Description
<table> यह एक table को परिभाषित करता है ।
<tr> यह table में एक पंक्ति (row) को परिभाषित करता है ।
<th> यह table में header cell को परिभाषित करता है ।
<td> यह table में एक cell को परिभाषित करता है ।
<caption> इसका उपयोग table का caption देने के लिए होता है |
<thead> इसका उपयोग table मे header content को एक समूह मे करने के लिए होता है |
<tbody> इसका उपयोग table मे body content को एक समूह मे करने के लिए होता है |
<tfoot> इसका उपयोग table मे footer content को एक समूह मे करने के लिए होता है |
