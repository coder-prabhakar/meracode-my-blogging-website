HTML Language in Hindi |

HTML एक ऐसी भाषा है जिसका इस्तेमाल website बनाने के लिए किया जाता है और उसे रंग रूप देने के लिए यानि उसे सुंदर बनाने के लिए CSS का इस्तिमाल होता है । HTML बहुत तरह के element से मिलकर बना होता है जिनका का काम web page पे अलग - अलग सामग्री को आलग - अलग तरीको से प्रदर्शीत करना होता है | यह अन्य भाषाओं जैसे C, C++, JAVA, Python आदि से बहुत आसान है इसे कोई भी व्यक्ति बहुत आसानी से सिख सकता है | कोई भी website बनाने मे मुख्य रूप से तीन भाषाओं का उपयोग होता है ये तीन भाषाए HTML, CSS और JavaScript है |

HTML Meaning in Hindi |

HTML का मतलब यानि इसका पूरा नाम Hypertext Markup Language होता है इसको हम छोटे नाम से पुकारते है जिसे HTML कहते है |

HTML का कार्य क्या है ?

HTML का कार्य web page पे सामग्री को प्रदर्शीत करना होता है | कोई भी website बनाने मे मुख्य रूप से तीन भाषाओं का उपयोग होता है ये तीन भाषाए HTML, CSS और JavaScript है | HTML का काम जो content, picture, video आदि website पर दीखता है वो सब Html मे लिखा जाता है, CSS का काम उसे सुन्दर बनाना है और JavaScript को website का दिमाग(mind) बोलते है |

HTML का आविष्कार कब हुआ ?

HTML का आविष्कार Tim Berners-Lee ने सन 1980 में जेनेवा में किया था | बर्नर्स ली ने ही 1989 में WWW (World Wide Web) का आविष्कार किया था । अभी के समय में HTML के विकास का पूरा जिमा “World Wide Web Consortium (W3C)” के पास है यही समय समय पर HTML मे सुधार करते रहते है ।

HTML का उपयोग कैसे करें ?

HTML का code लिखने के लिए आप कोई भी Text editor का उपयोग कर सकते है जैसे- Notepad , TextEdit आदि |

आपने जो भी code लिखा है उसे index.html के नाम से save करे आप चाहे तो अपने file का दूसरा नाम भी रख सकते है लेकिन ध्यान दे की आप जो भी नाम रखे उसके बाद मे .html लिखा होना चाहिए | जैसे - firstProject.html , myProject.html आदि |

कोई भी text editor मे आप जो भी कुछ लिखकर save करते है तो वो एक साधारण text file बनता है लकिन जब आप उस text file के नाम के अंत मे .html लिख कर save करते है तो वो एक HTML file बनता है जिससे browser को पता चलता है की इसके अंदर HTML का code है जिसे web page पे प्रदर्शीत करना है |

HTML लिखने का तरीका |

HTML मे कोई भी काम element के अन्दर होता है | HTML मे element के अंदर भी element हो सकता है | एक element दो tag से बना होता है opening tag त्तथा closing tag | निचे कुछ element दिए गए है जिसे देख कर आप समझ सकते है की element और उनके opening tag त्तथा closing tag क्या होते है |

Opening tag Closing tag
<html> </html>
<head> </head>
<body> </body>
<h1> </h1>
<p> </p>

निचे कुछ महत्वपूर्ण element के बारे मे जानकारी दि गई है |

<html> :- यह HTML का मुख्य element है जो दर्शता है की यह HTML का code है | HTML code मे पहला element यही आता है बाकी सारे HTML element इसके अंदर ही होते है |

<head> :- <html> element के अंदर दो ही element होते है जिसमे से पहला <head> है इसमें web page की मुख्य चिझे सामिल होती है | <head> element के अंदर जो code लिखा जाता है वो website पे नहीं दीखता है |

<body> :- <html> element के अंदर दूसरा element <body> होता है | हमें website मे जो कुछ भी दीखता है उन सारे का code <body> element के अंदर लिखा होता है |

<h1> :- <h1> element का उपयोग हम Heading यानि शीर्षक लिखने के लिए करते है |

<p> :- <p> element का उपयोग हम किसी भी प्रकार का paragraph यानि text लिखने के लिए करते है |

HTML के उदाहरण -

निचे एक उदाहरण दिया गया है जिसमे HTML का code दिया गया है जिसे देखने के बाद आपको अच्छे से इसके बारे मे जानकारी हो जाएगी |

<html>


<head>

<title>Page Title</title>

</head>


<body>

<h1>This is a Heading</h1>

<p>This is a paragraph.</p>

</body>


</html>
